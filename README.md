# 영화 서비스앱
## 프로젝트 소개 😎
React를 활용한 영화 서비스앱 구현 프로젝트 입니다.

## 이곳은 공개 Repo입니다.
1. 이 repo를 fork한 뒤
2. 여러분의 개인 Repo에서 프로젝트를 진행합니다. 
3. 개발이 끝나면 이 Repo에 PR을 보내어 제출을 완료해주세요.
4. 멘토님의 코드리뷰를 받습니다.
